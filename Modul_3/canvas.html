Browser tidak mendukung Canvas.

Kesimpulan Canvas:
Tag canvas menyediakan area gambar berbasis piksel yang dikendalikan sepenuhnya oleh JavaScript.
Berbeda dengan SVG yang menggunakan elemen XML, canvas menggambar melalui perintah JavaScript seperti fillRect(),
arc(), lineTo(), dsb.

Canvas cocok untuk animasi, game, grafik real-time, dan manipulasi gambar.
Hasil gambar canvas tidak bisa di-inspect seperti elemen HTML biasa
karena semuanya dirender sebagai piksel di dalam elemen tunggal.
