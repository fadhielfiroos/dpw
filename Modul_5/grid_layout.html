Kesimpulan Skenario 1:
Property display: grid mengubah elemen menjadi grid container.
grid-template-columns: auto auto auto membagi container menjadi 3 kolom dengan lebar otomatis (masing-masing 1/3 lebar container).
Item grid langsung mengisi kolom dari kiri ke kanan, lalu turun ke baris berikutnya secara otomatis.

Kesimpulan Skenario 3:
Property grid-column-gap mengatur jarak horizontal antar kolom, sedangkan grid-row-gap mengatur jarak vertikal antar baris. Semakin besar nilai px yang diisi, semakin lebar celah/jarak antar item grid. Kedua property ini berguna untuk memberi ruang antar sel grid.

Kesimpulan Skenario 4: Property grid-gap adalah shorthand yang menggabungkan grid-row-gap dan grid-column-gap menjadi satu deklarasi. Dengan nilai tunggal grid-gap: 50px, jarak antar baris DAN kolom sama-sama 50px. Ini lebih ringkas daripada menulis dua property terpisah. Nilai bisa juga dua angka: grid-gap: 30px 60px (row gap 30px, column gap 60px).
