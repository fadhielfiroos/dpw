Skenario 2



Kesimpulan:
Tanpa atribut name, setiap radio button berdiri sendiri sehingga ketiganya bisa dipilih sekaligus.
Jika semua diberi name yang sama , hanya satu yang bisa dipilih dalam satu waktu karena browser mengelompokkannya.
Jika setiap radio diberi name yang berbeda, masing-masing berdiri sendiri lagi sehingga semuanya bisa dipilih bersamaan dan mirip saat tanpa atribut name.

Skenario 3



Kesimpulan:
Berbeda dengan radio, checkbox bisa memilih lebih dari satu pilihan sekaligus.
Atribut name pada checkbox tidak membatasi jumlah pilihan.

Skenario 4




Kesimpulan:
type="color"menampilkan color-picker sehingga pengguna dapat memilih warna secara visual.
type="date" menampilkan kalender untuk memilih tanggal.
type="button" dengan atribut onclick menjalankan fungsi JavaScript saat tombol diklik.

Skenario 5




Kesimpulan:
type="datetime local" memungkinkan pemilihan tanggal dan waktu sekaligus.
type="month" hanya memilih bulan dan tahun tanpa tanggal spesifik.
type="email" memvalidasi format email secara otomatis saat form di-submit.

Skenario 6




Kesimpulan:
type="image" menggunakan gambar sebagai tombol submit.
type="file"membuka dialog untuk memilih file dari perangkat pengguna.
type="hidden" tidak terlihat di halaman namun tetap mengirim data ke server (berguna untuk menyimpan ID atau token tersembunyi).

Skenario 7



Kesimpulan:
type="number" menampilkan field angka dengan tombol naik/turun; atribut min dan max membatasi rentang nilai yang valid.
Jika min diubah menjadi -50>, nilai negatif menjadi diperbolehkan.
type="range" menampilkan slider dan nilai tidak terlihat langsung oleh pengguna sehingga cocok untuk pengaturan yang tidak memerlukan nilai presisi.

Skenario 8






Kesimpulan:
type="url" memvalidasi format URL (harus diawali http:// atau https://).
type="search" mirip text biasa namun pada beberapa browser menampilkan tombol hapus (×) dan desain khusus kolom pencarian.
type="tel" menampilkan keypad numerik di perangkat mobile,

Skenario 9



Kesimpulan:
type="time" memungkinkan pemilihan jam dan menit dengan format 24-jam atau 12-jam tergantung pengaturan browser/sistem.
type="week" memungkinkan pemilihan minggu dalam tahun tertentu (misal: Week 12, 2024).
Kedua tipe ini belum sepenuhnya didukung oleh semua browser, Safari misalnya menampilkan input teks biasa untuk beberapa tipe ini.
