Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan:
Tombol Reset mengembalikan semua isian ke nilai awal value yang didefinisikan di HTML, bukan mengosongkan field secara total.
Tombol Kirim mengirim data ke URL yang ditentukan pada atribut action.
Tanpa atribut method, default-nya adalah GET dan data muncul di URL sebagai query string
Dengan method="post", data dikirim di body HTTP sehingga tidak terlihat di URL, lebih aman untuk data sensitif.
Atribut target="_blank" membuka hasil di tab baru, sedangkan target="_self" (default) Jika membuka di tab yang sama.
