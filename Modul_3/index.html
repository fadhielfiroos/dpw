Ini adalah halaman beranda website. Halaman ini menggunakan elemen-elemen semantik HTML5 seperti header, nav, section, article, dan footer.

Perbedaan sebelum dan sesudah CSS ditambahkan sangat terlihat jelas. Tanpa CSS, semua elemen ditampilkan secara vertikal berurutan tanpa layout. Setelah CSS ditambahkan, nav dan article tampil berdampingan (float), header dan footer berwarna, serta teks menjadi lebih rapi dan terstruktur.
